# HeyDoc HIS

HeyDoc is een cloud-native huisartsinformatiesysteem, gebouwd op FHIR R4 en draaiend op Google Cloud Platform. Het systeem is ontwikkeld door een huisarts die geen geschikt alternatief vond, en is in productief gebruik in een huisartsenpraktijk in Leiden.

## Stack

- **FHIR R4** via Google Cloud Healthcare API
- **Cloud Functions** 
- **React** frontend

NEN 7510:2024 gecertificeerd (Kiwa K-0228563/1). Nictiz OID: 2.16.840.1.113883.2.4.3.245.

## Status

In actief gebruik als productiesysteem. Nictiz kwalificatietraject lopend.

## AI

Spreekuur.ai is online als onderzoeksomgeving voor AI-toepassingen in de huisartsenpraktijk. Eigen AI-modellen zijn nog niet in productie genomen.
